# BetterMail logo pack

- `01-source/`: traced SVG silhouette, approved 1024px extraction, and approved wordmark.
- `02-web/`: favicons, PWA icons, manifest, and HTML tags.
- `03-github/`: avatar, repository social preview, README wordmark.
- `04-desktop/`: Windows ICO/MSIX, transparent Linux icons, macOS ICNS/source sizes.
- `05-microsoft-auth/`: Entra square logos, banner logo, and dialog icon.
- `06-banners/`: social, web, and documentation banners.
- `07-future-mobile/`: Apple and Android source assets.
- `08-screen-shots/`: Screen shots of the app (redacted)

Primary `#615FFF`; secondary `#157EFB`; dark `#0E0F14`. Keep 12% clear space. Use monochrome for tray/status surfaces.
