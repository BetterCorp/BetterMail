# BetterMail support

Use [GitHub Issues](https://github.com/BetterCorp/BetterMail/issues/new/choose) for bug reports, feature requests, and installation or configuration help.

GitHub Issues are public. Remove names, email addresses, subjects, message bodies, tenant identifiers, access tokens, client secrets, and other personal or organizational data before posting. Use BetterMail's **Copy error** action, then redact the copied text.

For a bug, include the BetterMail version, operating system, exact reproduction steps, expected result, and actual result. Search existing issues first so related reports remain together.
